## 🐛 Common Issues & Fixes

**❌ Executor fails to inject / attach**
- Cause: Roblox was already running when Xeno launched, or Visual C++ Redistributable is missing.
- Fix: Close Roblox entirely, open `XenoLoader.exe` first, wait for setup to finish, then launch Roblox. If the error persists, install or update the Visual C++ 2019 Redistributable.

**❌ Android APK fails to install ("Parse Error")**
- Cause: "Install from Unknown Sources" is disabled, or the APK file was corrupted during download.
- Fix: Enable Unknown Sources in Android Security settings, then re-download the **xeno executor apk latest** version from this repository and try again.

**❌ IPA disappears on iPhone after a few days**
- Cause: Free AltStore certificates expire after 7 days — this is an Apple limitation, not a Xeno bug.
- Fix: Reconnect your device to AltStore on your PC and re-sign the IPA file. This takes under one minute.

**❌ Windows SmartScreen blocks the .exe**
- Cause: SmartScreen flags unsigned executables heuristically.
- Fix: Click "More info" → "Run anyway" in the SmartScreen dialog, or add the Xeno folder to Windows Defender exclusions before extraction.

**❌ Script Hub shows blank / won't load**
- Cause: Firewall or VPN is blocking Xeno's outbound network requests.
- Fix: Temporarily disable your VPN, whitelist Xeno in your firewall settings, and retry.

**❌ Roblox crashes immediately after injection**
- Cause: The installed Xeno build is outdated relative to the current Roblox client version.
- Fix: Download the latest version from this repository. Do not use cached or older builds after a Roblox update.

**❌ Script executes but has no visible effect in-game**
- Cause: The script may not support the specific game you're playing, or the game has server-side protections blocking certain client actions.
- Fix: Try a different script from the Hub specifically tagged for your game, or check the script thread on the Xeno Reddit community for known compatibility notes.
